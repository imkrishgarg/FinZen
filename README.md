# 🚀 FinZen — AI-Powered Financial Intelligence Platform

> 🇮🇳 Built for Indian Gen-Z — Simplifying Finance with AI, Security, and Intelligence

---

## 📌 Problem Statement

India’s Gen-Z faces a **fragmented, confusing, and risky financial ecosystem**:

* 🚨 **UPI Fraud** — Fake QR codes & scam UPI IDs
* 📄 **Insurance Complexity** — Hidden clauses & unreadable policies
* 🧩 **Too Many Apps** — 5–7 apps for finance management
* 📉 **Low Financial Literacy** — Poor investment decisions

---

## 💡 Our Solution — FinZen

FinZen is a **unified AI-powered financial platform** that replaces multiple apps with one intelligent system.

### 🔥 Core Features

| Feature                | Description                                       |
| ---------------------- | ------------------------------------------------- |
| 🔐 UPI Fraud Shield    | AI trust score (0–100) for UPI IDs & QR codes     |
| 📄 Insurance Decoder   | Upload PDF → AI explains coverage & risks         |
| 📊 Market Intelligence | Real-time stock & crypto insights with AI signals |
| 🎙️ Voice AI Copilot   | Ask financial questions via voice/chat            |
| 🗺️ Smart Map          | Find ATMs, banks & fraud-prone zones              |
| 🧪 Investment Sandbox  | Simulate investments before investing             |

---

## 🧠 How It Works

1. User inputs data (UPI / PDF / query)
2. Backend processes via APIs + AI
3. GPT-based intelligence generates insights
4. UI displays results with charts & explanations

## ⚙️ Local Setup Guide

### 1️⃣ Clone Repository

```bash id="6x2h8s"
git clone https://github.com/your-username/finzen.git
cd finzen
```

### 2️⃣ Install Dependencies

```bash id="n3d9fk"
npm install
```

### 3️⃣ Setup Environment Variables

Create `.env.local`:

```env id="m8d3qh"
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=

MONGODB_URI=

OPENAI_API_KEY=

COINGECKO_API_KEY=
FINNHUB_API_KEY=

MAPS_API_KEY=
```
